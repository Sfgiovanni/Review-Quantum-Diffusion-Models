# Limitations

- IEEE and Springer records are Crossref metadata retrieved by DOI-prefix restrictions, not direct proprietary API searches.
- Crossref abstracts can be missing, truncated, or contain publisher-specific markup.
- DOI-prefix scoping can miss records with other relevant publisher prefixes.
- Metadata can change over time.
- Automated scoring and classification are only prioritization aids.
- Final inclusion requires human screening.
