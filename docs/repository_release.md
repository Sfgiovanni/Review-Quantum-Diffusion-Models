# Repository Release

Use semantic versioning. For the first public archive, tag `v0.1.0`.

1. Confirm tests, lint, notebook execution, and smoke test pass.
2. Confirm no credentials or protected PDFs are present.
3. Push to GitHub.
4. Connect the repository to Zenodo.
5. Create a GitHub release.
6. Update `CITATION.cff` with the Zenodo DOI after archival.
